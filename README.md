\# FlexFlix



A streaming service for books and music — think Netflix, but for readers

and music lovers. Discover, stream, and save your favorite books and

tracks in one place.



\## Features



\- Browse a catalog of books and music by genre, author, or mood

\- Add content to personal playlists

\- Search and filter across both media types

\- Lightweight JSON-based data layer (no heavy DB required)



\## Tech Stack



\- Python 3.11

\- Flask (web framework)

\- JSON (data storage)

\- pytest (testing)



\## Getting Started



```bash

\# Clone the repo

git clone https://github.com/wayf-fert/FlexFlix.git

cd FlexFlix



\# Install dependencies

pip install -r requirements.txt



\# Run the app

python app/main.py

```
